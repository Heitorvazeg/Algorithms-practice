# 🧠 Algorithms Practice

Bem-vindo ao repositório de **prática de algoritmos**! Aqui você encontrará implementações, anotações e exercícios para treinar lógica e estruturas de dados visando:

- 👨‍💻 Entrevistas técnicas
- 🏆 Competições de programação (como Maratona SBC, ICPC, etc.)
- 📈 Aprimoramento de habilidades em resolução de problemas

---

## 📂 Estrutura do Repositório

```
algorithms-practice/
├── 📁 algorithms/       # Implementações de algoritmos clássicos (ordenadores, buscas, recursivos, etc.)
├── 📁 data-structures/  # Estruturas de dados como pilhas, filas, listas, árvores, grafos, etc.
├── 📁 leetcode/         # Soluções e anotações de problemas resolvidos na plataforma LeetCode
├── 📁 maratonas/        # Problemas e soluções de competições como Maratona SBC, OBI e afins
└── README.md            # Arquivo principal de descrição do projeto
